#  Project Overview

This project analyzes groundwater quality using Power BI and Python.

The objective is to monitor water quality parameters, identify unsafe drinking water, compare state-wise conditions, and generate meaningful business insights through interactive dashboards.



# Objectives

✔ Monitor pH levels

✔ Analyze Conductivity

✔ Study Temperature Trends

✔ Identify Drinkable Water

✔ Compare State-wise Performance

# Data Cleaning

Python was used to

- Merge multiple CSV files
- Remove duplicates
- Handle missing values
- Standardize State Names
- Calculate Average pH
- Create Drinkability Categories
- Generate Temperature Bands

---

# Key Insights

- 90% of water samples are drinkable.
- Average pH remains within WHO standards.
- Higher temperatures slightly reduce pH.
- Rajasthan records the highest average pH.
- A small percentage of samples are acidic.
